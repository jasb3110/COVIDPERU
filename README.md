# COVIDPERU
The COVIDPERU is project which have goal to analyze COVID´s data set on Peru. Before that whole information have downloaded government source of Peru. then it had cleared of mistakes and to order.

This data include date since 6/06/2020 to 21/02/2022(Column 1), COVID positive rate (Column 2), COVID death index which estimated of excess historical (SINADEF-column 6), free UCI bed in percent(Column 10) and vaccinated people which have 1 to 5 doses (Column 12 to 16).
